
# PeerX AI

PeerX AI is an AI-powered multi-agent platform designed to automate academic peer review.

## 🚀 Problem
Academic publishing suffers from reviewer shortage, delays, and biased evaluations, especially in developing countries.

## 💡 Solution
PeerX AI uses multiple intelligent agents to generate accurate, structured, and unbiased research feedback.

## 🔧 Features
- Reviewer Agent (analyzes methodology and quality)
- Meta-Reviewer Agent (summarizes and scores)
- Bias Detection Agent
- Explainable AI feedback
- Retrieval-Augmented Generation (RAG)

## 🛠️ Tech Stack
- Python
- FastAPI
- LangGraph
- LLMs
- FAISS Vector Database
- React (Frontend)

## 📊 Status
Currently in development (Final Year Project → Startup)

## 🌍 Vision
To democratize high-quality research feedback globally.
